Tensor Decompositions

  • C
  • Convex Optimization Theory
  • Multilinear Algebra

A major focus of my research at Reservoir Labs is the theory and applications of tensor decompositions. This is an unsupervised learning technique that finds patterns of activity in multidimensional data, and captures more cross-variable correlations than conventional matrix decomposition techniques. I implemented an algorithm called CP-APR-PQNR, which is a tensor decomposition algorithm which uses second order methods (L-BFGS) to fit a tensor decomposition for the special case of sparse count data tensors. I also developed an algorithm known as CP-WOPT which predicts missing entries of a tensor, which has useful applications in prediction and anomaly detection.

VINAS

  • C++
  • R
  • Machine Learning
  • Statistics
  • Data Visualization

This project holds a special place in my heart because it was my first large scale machine learning project. This was a research effort at the Gerstein Lab at Yale, with mentorship from Dr. Jing Zhang, with the purpose of designing a machine learning algorithm to find statistically significant genes that were linked with cancer. I coded an end-to-end machine learning pipeline which operated on large data (chromosome data on the order of terabytes), and had to implement a variety of computer science tricks, such as efficient sorting and hashing of big data, machine learning methods, such as logistic regression and PCA, and statistics and theoretical math to compute p-values for the gene with formal justification.
